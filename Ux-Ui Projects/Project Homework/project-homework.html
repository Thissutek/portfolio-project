Project Overview

Project Homwework is an online dance education platformed designed to cultivate the growth of creativity within dancers. As a website redesign I've changed a lot of the layout and user flows to make it more seamless and easier for a general audience directing it towards dancers from different levels and backgrounds.

Role

UX/UI Designer

Timeline

12 Weeks

Tools

Figma

Establishing Brand

Establishing their already made brand by creating their design systems and using color guide and mood boards to create connection with the brand. Establishing the brand gave a better scope of the project knowing what I can work with and their values I have to re-establish within their brand.
